 1 уровень сложности: 1. Как скопировать на свой компьютер репозиторий из github.com?
1.Чтобы клонировать репозиторий с помощью ключа SSH, нужно скопировать ключ.
2. Затем открыть Git Bash.
3.Изменить текущий рабочий каталог на то место, где мы хотитим клонировать каталог.
4.Ввести git clone и вставьте URL-адрес, скопированный ранее.



2.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

Если у нас уже есть проект в каталоге, который не находится под версионным контролем Git, то сначала нужно перейти в него. 
-cd C:/Users/user/my_project
-git init
-git add .
-git commit -m 'комментарий'

3.Как получить на локальный компьютер изменения с github.com
-git pull origin[branch name]

Какую информацию нам дают команда :
git status - проверка статуса
git log - просмотр изменений коммитов
git remote -v - эта команда позволяет создавать, просматривать и удалять подключения к другим репозиториям.


4.Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?
 - git checkout [file name]
5.Как вернуть файл из индекса в рабочий каталог?
-git reset[file name]
6.Как отменить ошибочный commit ?
-git checkout[file name]
7.Как исправить коммит отправленный на сервер GitHub?
-git commit --amend
-git add forgotten_file
